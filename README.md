# PYMES Administrativo
Sistema de gestión comercial para pequeños negocios (inventario, precios/costos, pedidos/ventas).

## Stack
- Backend: NestJS + TypeScript + PostgreSQL
- Frontend: React + TypeScript + Tailwind/Vite
- Infra: Docker + AWS/Vercel (despliegue)

## Instalación
1. `git clone https://github.com/[julianandreshf]/pymes-administrativo`
2. `cd pymes-administrativo`
3. `npm install` (raíz)
4. `docker-compose up` (entorno dev)
